Crimes in London Metropolitan Police Areas

This project analyzes crime trends within the London Metropolitan Police Areas, excluding the City of London, which is policed separately. The primary objective is to extract, process, and visualize crime statistics to provide actionable insights. Clustering Crime Data in Los Angeles

This project analyzes crime data in Los Angeles, focusing on property and violent crimes, using clustering techniques to identify patterns.
